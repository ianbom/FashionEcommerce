# Split PRD Admin Dashboard E-Commerce

File-file ini adalah pecahan PRD Admin Dashboard agar dapat dikerjakan bertahap oleh Codex tanpa memecah fitur yang saling bergantung.

## Urutan pengerjaan

1. `00-master-implementation-plan.md`
2. `01-foundation-auth-layout-settings.md`
3. `02-catalog-product-variant-stock.md`
4. `03-sales-order-payment-shipment.md`
5. `04-customer-marketing-content.md`
6. `05-dashboard-reports-audit.md`

## Prinsip penting

- Product, variant, image, category, collection, dan stock dikerjakan dalam satu paket.
- Order, payment, shipment, tracking, dan webhook dikerjakan dalam satu paket.
- Dashboard/report dikerjakan terakhir setelah data utama tersedia.
